# **App Name**: HireGenius AI

## Core Features:

- Resume Intelligence: Automatically parse resumes to extract skills, experience, projects, and tools. Identify and flag potential duplicate or fake resumes. Output a structured, searchable candidate profile.
- Skill Graph & Matching: Match candidate skills with job role requirements using a comprehensive skill graph. Calculates a skill fit percentage and experience fit score for each candidate.
- AI-Powered Screening: Generate role-based screening questions automatically using AI, assess candidate answers based on predefined criteria.
- Interview Intelligence: Provide AI support during interviews by summarizing key points from candidate responses. Automatically summarize feedback collected after the interview.  
- Smart Scheduling: Automate interview scheduling by identifying optimal time slots based on interviewer and candidate availability.
- Personalized Learning: Provide personalized learning recommendations based on detected skill gaps identified during initial assessment.
- Central Intelligence Layer: The core AI brain learns hiring patterns, recommends actions, predicts outcomes, and automates workflows. It uses the talent knowledge base as a tool. Powers candidate ranking, hire/no-hire decisions, attrition prediction, and recruiter efficiency analytics.

## Style Guidelines:

- Primary color: A calming blue (#6699CC) to instill confidence and reliability.
- Background color: Light grayish-blue (#F0F4F8) for a clean and professional feel.
- Accent color: Soft violet (#9966CC) for highlighting key actions and insights.
- Body font: 'PT Sans', a humanist sans-serif combining modernity with warmth, suited for both headlines and body.
- Clean and professional icons representing different skills, assessments, and data insights.
- Clean and structured layout with clear information hierarchy. Focus on data visualization to present insights effectively.
- Subtle transitions and animations to enhance user experience and guide them through the hiring process.
